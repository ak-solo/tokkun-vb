# 4章 繰り返し

## 基礎知識

### For/Next ループ

回数が決まっているときは `For` ループを使います。

```vbnet
For i = 1 To 5
    Console.WriteLine(i)   ' 1, 2, 3, 4, 5
Next

' Step で増減幅を変更できる
For i = 0 To 8 Step 3
    Console.WriteLine(i)   ' 0, 3, 6
Next
```

---

### While ループ

条件が真の間くり返すときは `While` を使います。

```vbnet
Dim n As Integer = 1
While n <= 5
    Console.WriteLine(n)
    n += 1
End While
```

---

### Do/Loop

`Do` ループは最低 1 回は実行されます（ループ末尾で条件を評価）。

```vbnet
Dim n As Integer = 0
Do
    n += 1
    Console.WriteLine(n)
Loop While n < 5     ' 条件が真の間くり返す
```

ループ先頭で条件を評価したい場合：

```vbnet
Do While n < 5
    ' ...
Loop
```

---

### Exit For / Continue For

`Exit For` でループを途中で抜け、`Continue For` で次の反復にスキップできます。

```vbnet
For i = 1 To 10
    If i = 6 Then Exit For       ' 6 に達したら終了
    If i Mod 2 = 0 Then Continue For  ' 偶数はスキップ
    Console.WriteLine(i)         ' 1, 3, 5
Next
```

---

### 累計・最大・最小

```vbnet
Dim numbers = {5, 3, 8, 1, 9}
Dim sum As Integer = 0
Dim max As Integer = numbers(0)
Dim min As Integer = numbers(0)

For Each n In numbers
    sum += n
    If n > max Then max = n
    If n < min Then min = n
Next
```

---

### ネストしたループ（二重ループ）

ループの中にループを入れると、二次元的な処理ができます。

```vbnet
For i = 1 To 3
    For j = 1 To 3
        Console.Write($"{i * j} ")
    Next
    Console.WriteLine()
Next
```

### 問題 4-15

**素因数分解**とは、整数を素数の積に分解することです。

```
12  = 2 × 2 × 3
20100 = 2 × 2 × 3 × 5 × 5 × 67
```

2 以上の整数 `n` を受け取り、素因数分解した結果を昇順の `Integer` 配列で返す関数を実装しなさい。

例: `n=20100` → `{2, 2, 3, 5, 5, 67}`

**ヒント:** 小さい数（2, 3, 4, …）から順番に試し割りしていきます。`n` が割り切れる間はその数を記録して `n` を割り続け、割り切れなくなったら次の数へ進みます。

### 問題 4-16

**素数**とは、1 より大きく、1 と自分自身以外では割り切れない整数のことです。

```
素数:    2, 3, 5, 7, 11, 13, 17, 19, ...
素数でない: 4 (= 2×2), 6 (= 2×3), 9 (= 3×3), ...
```

`Integer` 型引数 `n` を受け取り、`n` が素数なら `True`、そうでなければ `False` を返す関数を実装しなさい。

**ヒント:** `n` が素数かどうかは、2 から √n までの整数で割り切れるか調べれば判定できます。たとえば `n=100` なら 2〜10 の範囲だけ確認すれば十分です（`Math.Sqrt(n)` で √n を求められます）。

### 問題 4-22

**フィボナッチ数列**とは、直前の 2 つの項を足し合わせることで次の項を作る数列です。

```
0, 1, 1, 2, 3, 5, 8, 13, 21, ...
↑  ↑
0 + 1 = 1
   1 + 1 = 2
      1 + 2 = 3  ...
```

フィボナッチ数列の 1000 以下の項を `Integer` 配列で返す関数を実装しなさい。

最初の 2 項は `0`、`1` とする。

例: `{0, 1, 1, 2, 3, 5, 8, 13, 21, …, 987}`

**ヒント:** 変数を 3 つ（`a`、`b`、`nxt`）用意して、`nxt = a + b` → `a = b` → `b = nxt` と更新していきます。
