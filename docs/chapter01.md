# 1章 表示・変数・演算

## 基礎知識

### 画面への出力

`Console.WriteLine` は引数を表示して改行します。`Console.Write` は改行しません。

```vbnet
Console.WriteLine("Hello World")   ' Hello World と表示して改行
Console.Write("Hello")             ' Hello と表示（改行なし）
Console.Write(" World")            ' Hello World となる
```

文字列の中に変数の値を埋め込むには、`$"..."` （文字列補間）が使えます。

```vbnet
Dim x As Integer = 10
Console.WriteLine($"x={x}")        ' x=10 と表示
Console.WriteLine($"x={x},y={5}")  ' x=10,y=5 と表示
```

---

### 変数の宣言と代入

`Dim 変数名 As 型名` で変数を宣言します。宣言と同時に値を代入できます。

```vbnet
Dim x As Integer        ' Integer 型の変数 x を宣言
Dim y As Integer = 5    ' 宣言と同時に 5 を代入
x = 10                  ' x に 10 を代入
```

#### 主なデータ型

| 型名 | 内容 | 例 |
|---|---|---|
| `Integer` | 整数（−2,147,483,648 〜 2,147,483,647） | `42`, `-7` |
| `Double` | 倍精度浮動小数点数 | `3.14`, `-0.5` |
| `String` | 文字列 | `"Hello"` |
| `Boolean` | 真偽値 | `True`, `False` |

---

### 演算子

| 演算子 | 意味 | 例 | 結果 |
|---|---|---|---|
| `+` | 加算 | `3 + 2` | `5` |
| `-` | 減算 | `3 - 2` | `1` |
| `*` | 乗算 | `3 * 2` | `6` |
| `/` | 除算（結果は Double） | `10 / 3` | `3.333...` |
| `\` | 整数除算（商のみ） | `10 \ 3` | `3` |
| `Mod` | 剰余（余り） | `10 Mod 3` | `1` |
| `^` | 累乗 | `2 ^ 8` | `256` |

> **注意:** `Integer` 型の変数同士でも `/` を使うと結果は `Double` になります。
> 整数のまま割り算したい場合は `\` を使います。

## 練習問題

### 問題 1-1

`"Hello World"` という文字列を返す関数を実装しなさい。

**ヒント:** `Return "Hello World"` と書くだけです。

---

### 問題 1-2

`Integer` 型の変数 `x` に数値 `11` を代入し、`"x=11"` という文字列を返す関数を実装しなさい。

**ヒント:** `$"x={x}"` のように文字列補間を使いましょう。

---

### 問題 1-3

`Integer` 型の変数 `x`、`y` に数値 `13`、`17` を代入し、`"x=13,y=17"` という文字列を返す関数を実装しなさい。

---

### 問題 1-4

`Integer` 型の変数 `x` に `13` と `17` の和を代入し、`x` の値（整数）を返す関数を実装しなさい。

**ヒント:** `Dim x As Integer = 13 + 17` のように書いて `Return x` します。

---

### 問題 1-5

`13` と `17` の積（整数）を返す関数を実装しなさい。ただし、**変数を使わずに** `Return` の式の中で演算すること。

---

### 問題 1-6

次の手順で計算し、途中の値を `"21,10"` のようなカンマ区切りの文字列で返す関数を実装しなさい。

1. `Integer` 型の変数 `x` に `7` を代入する
2. `x` の値を 3 倍にして `x` に代入する
3. `x` を 2 で整数除算した値を求める
4. ステップ 2 の結果とステップ 3 の結果をカンマ区切りで返す

**ヒント:** `x` を 3 倍した後、`$"{x},{x \ 2}"` のように文字列補間でまとめて返せます。

---

### 問題 1-7

`Integer` 型の変数 `x`、`y` にそれぞれ `3`、`7` を代入し、`x` と `y` の値を**入れ替えて**、`"x=7,y=3"` という文字列を返す関数を実装しなさい。

**ヒント:** `x` と `y` 以外に、もうひとつ変数が必要です。

---

### 問題 1-8

`Integer` 型の変数 `x`、`y` に `19`、`23` を代入し、その積を変数 `z` に代入して `z` の値（整数）を返す関数を実装しなさい。

---

### 問題 1-9

`Integer` 型の引数 `x` を受け取り、`x` を **2 倍・3 倍・4 倍** した結果をカンマ区切りの文字列で返す関数を実装しなさい。

例: `x = 3` のとき `"6,9,12"` を返す

---

### 問題 1-10

`Integer` 型の引数 `x` を受け取り、`x` の **1 乗・2 乗・3 乗** をカンマ区切りの文字列で返す関数を実装しなさい。

例: `x = 2` のとき `"2,4,8"` を返す

**ヒント:** 累乗は `^` 演算子を使います（例: `x ^ 2`）。

---

### 問題 1-11

`Integer` 型の変数 `x`、`y` に `10`、`3` を代入し、`x / y` の結果（`Double`）を返す関数を実装しなさい。

**ポイント:** `Integer` 同士でも `/` 演算子を使うと結果は `Double` になります（`3.333...`）。

---

### 問題 1-12

`Integer` 型の変数 `x` に `10 \ 3`（整数除算）の結果を代入し、`x` の値（整数）を返す関数を実装しなさい。

**ポイント:** 問題 1-11 と比べると、`/` と `\` の違いがよくわかります。

---

### 問題 1-13

2 つの `Integer` 型引数 `x`、`y`（`x` > `y`）を受け取り、`x ÷ y` の **商と余り** をカンマ区切りの文字列で返す関数を実装しなさい。

例: `x = 10, y = 3` のとき `"3,1"` を返す（商 3、余り 1）
